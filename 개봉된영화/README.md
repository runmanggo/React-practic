### 개봉된 영화 실습을 하면서 배운것
1) props 개념
2) react-router-dom 사용
   - react-router-dom에서 제공하는 몇몇 기능을 사용해봄
   - BrowserRouter : 라우팅을 위한 최상위 컴포넌트
   - Route : 특정 경로에 해당하는 컴포넌트를 렌더링
   - Link: 페이지 간의 전환을 위한 컴포넌트
   - useLocation : 현재 경로 정보를 가져오는 훅
   - useParams :  URL 경로 매개변수 값을 가져오는 훅으로, 동적인 경로에서 매개변수 값을 추출하는 데 사용
3) 실습 폴더에 이해를 위한 주석들이 있어서 전체적인 파일이 깔끔하지 않은것 주의!
4) 세부 내용은 벨로그에 업로드 예정입니다 : https://velog.io/@runmanggo
